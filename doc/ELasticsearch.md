## 文档、类型、索引

- Index
- Type
- Document

文档是Elasticsearch中的可以被索引的基本数据单位。文档必须属于一个类型。索引包含一个或多个类型。

对比与关系型数据库，索引类似于数据库，类型类似于表，文档类似于一行记录。
